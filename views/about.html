About EOS EZ Chat!

EOS EZ Chat is a simple Socket IO enabled chat room. To use, simply click the login button to link your Scatter account then type a message. You will see your message update as your transaction moves from (pending, confirmed, irreversible). This is an easy way of seeing how long it takes for a transaction to become irreversible. This app is currently hosted on the Jungle Testnet, so if you'd like to test it, you will need to create an account.
